最重要的属性：
	可见性， 尺寸， 精确定位
	堆叠次序， 透明度， 剪裁区域， 外边距， 内边距， 边框， 颜色

为了确定有边框元素在屏幕上的全尺寸，必须把元素的宽度加上左右两边的内边距和左右两个边框宽度，
把元素的高度加上上下两边的内边距和上下两个边框宽度。

position
top left：不是从容器内容区域的左上角开始度量的，而是从容器内边距的左上角开始的。
bottom right
width height
z-index	:指定元素相对于其他重叠元素的堆叠次序
display	：指定元素是否以及如何显示
visibility	：指定元素是否可见
clip	：定义元素的剪裁区域，只显示元素在区域内的部分
overflow	:指定元素比分配空间要大时的处理方式
margin border padding
background	:指定元素的背景颜色或图片
opacity	：指定元素的不透明度(或半透明度)

relative:在使用相对定位时，无论是否进行移动，元素仍然占据原来的空间。因此，移动元素会导致它覆盖其它框
absolute:它的定位要么是相对于最近定位的祖先元素，要么是相对于文档本身
设置为绝对定位的元素框从文档流完全删除，并相对于其包含块定位，
包含块可能是文档中的另一个元素或者是初始包含块。
元素原先在正常文档流中所占的空间会关闭，就好像该元素原来不存在一样。
元素定位后生成一个块级框，而不论原来它在正常流中生成何种类型的框。

块级元素会尽可能的在宽度上充满容器，而在高度上尽可能的小，够用就好了

把 display 设置成 none 不会保留元素本该显示的空间，但是 visibility: hidden; 还会保留。


在 CSS 中，width 和 height 指的是内容区域的宽度和高度。增加内边距、边框和外边距不会影响内容区域的尺寸，但是会增加元素框的总尺寸。
